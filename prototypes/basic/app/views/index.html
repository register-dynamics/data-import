Upload your monthly pension return

This demonstration service enables you to upload your monthly pension return as a single spreadsheet to show how your service could submit multiple records in a single transaction.

By uploading your pension returns on a monthly basis we are able to keep current data and future projections up to date, ensuring that you and your staff are kept as up to date as possible

To begin, you will need a spreadsheet or comma-separated file containing the previous month's employee salaries alongside their employee reference number and personal details. After uploading your file you will need to choose which sheet contains the data, and then you will need to tell us which fields in your spreadsheet are associated with the following fields:

  • Title
  • First name
  • Surname
  • Employee number
  • Employment start date
  • Salary (required)
  • Payment date

The Salary field is required. If you do not provide a salary in a particular row, an error message will be shown providing context for where the error occurs. Processing will continue, but the row will not be imported. You can then correct the error and re-upload the file.

The following sample files are provided to demonstrate the upload process. You can use these files to explore the service without needing to create your own data.

Sample file 1: pensions-data-sample.xlsx - an Excel spreadsheet containing 10 records with no errors.

Sample file 2: tribbles.xlsx - an Excel spreadsheet containing my favourite Star Trek aliens.

On completion, you will be provided with a reference number for the upload which should be quoted in any correspondence about this process.
