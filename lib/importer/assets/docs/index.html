About the Data Upload Design Kit

The Data Upload Design Kit is a free tool that offers a quick and easy way to gather data in bulk, from users. It works with the GOV.UK Prototype Kit and consists of a set of components and patterns for asking users for data as a spreadsheet.

These patterns and components are most useful when capturing a lot of information at once and when drawing from an existing source. They can be used as part of the GOV.UK Prototype Kit, in production as part of a real service and their designs are compatible with the GOV.UK Design System.

This kit is built and maintained by the team at Register Dynamics. For help using the tool please contact: hello@register-dynamics.co.uk.
